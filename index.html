Lucas Nazário - Desenvolvedor Full-Stack

Atuo como desenvolvedor Full Stack, com foco na construção de APIs REST utilizando Node.js, Express e TypeScript, integrando bancos de dados relacionais com Sequelize e não relacionais com MongoDB. No front-end, desenvolvo interfaces responsivas com React, React Native, HTML5, CSS3 e JavaScript. Tenho domínio na construção, arquitetura e manipulação de bancos de dados SQL e NoSQL, adequando-os conforme a necessidade do projeto. Utilizo Docker para garantir ambientes padronizados e isolados, e aplico boas práticas de versionamento, integração contínua e automação com Git e GitHub. Tenho domínio em todo o ciclo de desenvolvimento, da lógica de negócio à interface do usuário.
